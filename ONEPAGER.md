# AI 코딩 아레나 — 원페이저

**한 줄:** 바이브코딩 시대의 백준 — 프롬프트로 AI 봇과 겨루는 실시간 코딩 대전 게임.

| | |
|---|---|
| **라이브** | https://ai-coding-arena-live.vercel.app |
| **문제** | 알고리즘은 AI가 다 풀고 백준마저 종료(2026.04.28). 프롬프트 실력을 잴 곳·겨룰 무대·점수가 없다. |
| **해결** | 라이브 dev 사이트를 타깃으로 제한시간 내 프롬프트 재빌드 → gpt-4o 비전 채점 → AI 봇과 승부. |
| **차별점** | ① 난이도=실 AI 모델 차등(어려울수록 봇이 진짜 잘 만듦) ② 비전 심판(렌더 이미지 채점, 안티치트) ③ 게임화(XP·등급·VS연출·트래시토크) ④ AI vs AI(멀티 LLM 확장) |

### 게임 루프
모드(재현/자유)+난이도 선택 → 라이브 타깃 → VS 인트로 → **제한시간 내 무제한 재빌드(봇은 독립 빌드 1회)** → 타임아웃/제출 → gpt-4o 비전 루브릭+코칭 → 승부(콘페티/사운드) → 명예의전당 기록 + 공유.

### 난이도 = AI 모델 차등
| 난이도 | 시간 | 봇 | 캐릭터 |
|--------|------|-----|--------|
| 쉬움 | 120초 | 페이크(안정) | 🐣 코드뉴비 |
| 보통 | 90초 | gpt-4o-mini 실빌드 | 🏃 코드러너 |
| 어려움 | 60초 | gpt-4o 실빌드 | 🤖 코드마스터 |

### 임팩트
- **Learn** — 지면 AI가 프롬프트를 고쳐주고 재도전(놀며 느는 프롬프트 엔지니어링)
- **Viral** — 글로벌 랭킹·명예의 전당·OG 공유 카드로 자발적 확산
- **Scale** — 프롬프트 기본기 → 다음 시즌 '하네스 엔지니어링' 평가로 확장
